# xianyu-publish

闲鱼（goofish.com）发布与店铺基础运营自动化 Skill（OpenClaw）。

## 功能
- 发布新商品
- 编辑已发布商品（价格/描述）
- 下架/重新上架（含 `publish?itemId=` 兜底路径）
- 消息汇总
- 风控安全模式（默认）

## 快速开始
对助手说：`开始闲鱼托管`
