# 闲鱼 Skill 功能清单（在“发布”之外）

## P0（建议优先补齐）
1. 在架商品管理
   - 编辑：价格、描述、标签、发货方式
   - 状态：上架/下架
   - 删除商品
2. 发布结果追踪
   - 自动识别“发布成功/失败原因”
   - 失败分类（缺图、缺属性、认证拦截、风控）
3. 草稿能力
   - 保存草稿
   - 从草稿恢复并继续发布
4. 图片管理
   - 多图上传（首图+细节图）
   - 上传后顺序调整

## P1（实用增强）
1. 消息与咨询处理
   - 打开“消息”页，汇总未读咨询
   - 按商品维度整理买家问题
2. 批量发布
   - 模板化字段（统一发货、统一文案框架）
   - 多商品逐条发布 + 每条回执
3. 价格策略辅助
   - 同类商品粗对比（仅提示，不自动改价）
   - 一键建议“售价/原价”区间
4. 风险拦截
   - 发布前检查敏感词、违规词、联系方式违规写法

## P2（后续扩展）
1. 数据看板
   - 浏览量、咨询量、成交转化（人工可读汇总）
2. 自动回复草案
   - 按常见问法生成回复模板（不自动发送）
3. 多账号发布编排
   - 账号切换安全校验
   - 防串号确认

## 当前已验证能力（本轮）
- 发布页文案填写
- 价格填写（售价/原价）
- 发货方式与自提开关
- 属性选择（如年龄/性别）
- 图片上传（程序化注入 + 页面回显核验）
- 发布动作触发与成功落地页核验

## 仍需补测
- 编辑已发布商品的稳定性
- 删除/下架按钮的页面差异
- 认证拦截后的恢复流程
- 批量发多条时的限频与风控表现
